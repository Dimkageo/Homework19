Домашнє завдання
Мета цього домашнього завдання — створити повноцінний REST API для нотаток. API повинен бути побудований з використанням інфраструктури Spring Boot з модулями Spring Data, Spring MVC. Додаток має використовувати реляційну БД (in-memory H2 або іншу, наприклад PostgreSQL/MariaDB), а також мати мінімум одну міграцію з фреймворку для міграцій БД Flyway

API повинен мати можливість виконувати наступні дії:

Створення нотатки
Отримання нотатки
Оновлення нотатки
Видалення нотатки
Загальні вимоги
Використання Spring Boot
Використання модулів Spring Data, Spring MVC
Використання реляційної БД
Наявність однієї міграції з підключеного фреймворку Flyway з описаною таблицею нотаток
Підтримка CRUD операцій для нотаток
Наявність валідації полів запитів при CRUD операціях
